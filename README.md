## Three-projection-mapper

Experimenting with [Bernard Hoffman's super-cool three-projection-mapper plugin](https://github.com/bhoffmann93/three-projection-mapper).

[Check it out here](https://mikebelanger.github.io/three-projection-mapper-video/)

This example shows how a `<video>` tag can be incorporated into a THREE.js scene.

Various pieces of this were done with the assistance of [opencode](https://www.opencode.ai)
